# bft容错数计算推到

- 假设所有节点数位N,存在F个作恶节点，R个忠实节点，X为法定集合，则满足N=R+F
- 为保证结果不产生分歧，有唯一结果，则需法定集合大于一半以上的忠实节点通过，且法定集合中不包括作恶节点，则X-F>R/2 即 X>R/2+F.
- 为保证系统能够正常运行，法定集合不能大于忠实的节点数，X<=R,即R>=X
- => R>R/2+F
- => R>2F
- => N-F>2F
- => N>3F 即 N=3f+1
